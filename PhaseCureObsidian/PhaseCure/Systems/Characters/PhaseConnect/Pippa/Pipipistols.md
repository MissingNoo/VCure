## Description 
One pistol shoots in aimed direction, another at random targets

### Levels 
- Level 1:
	- [ ] Knockback
	- [x] One pistol shoots in aimed direction.
	- [x] Another at random targets.
- Level 2:

- Level 3:

- Level 4:

	- [x] Each bullet can pierce +1 times.
- Level 5:
	- [x] Increase damage by 50%.
- Level 6:
